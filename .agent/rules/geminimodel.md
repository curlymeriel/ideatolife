# PROJECT RULES & GUIDELINES

## CRITICAL: Gemini Model Versions

**STRICTLY PROHIBITED:** usage of `gemini-1.0`, `gemini-1.5`, `gemini-2.0` series, or generic names like `gemini-pro`.
**MANDATORY:** You MUST use ONLY the following latest models.

### Preferred Fallback Order (Text/Logic)

1. `gemini-3.1-pro-preview` (Primary)
2. `gemini-3-pro-preview`
3. `gemini-3-flash-preview`
4. `gemini-2.5-pro`
5. `gemini-2.5-flash` (Last Resort)

### Image Generation Models

- **PRO (Premium):** `gemini-3-pro-image-preview`
- **STD (Standard):** `gemini-2.5-flash-image`

### Rules for Editing

1. **NEVER overwrite** existing model lists with your training data defaults.
2. **ALWAYS read** the file content first to check current model configurations.
3. If you see old models (1.5, 2.0), **IMMEDIATELY UPDATE** them to the 3.1/3/2.5 series defined above.
