# PPT Templates & Resources

이 폴더는 "Idea to Life" 프로젝트에서 사용할 PPT 서식 샘플과 관련 리소스를 보관하는 장소입니다.

## 📁 구성 가이드
- **templates/**: `.pptx` 또는 `.potx` 서식 파일을 넣어주세요.
- **references/**: 레이아웃 참고용 스크린샷(`png`, `jpg`)이나 디자인 가이드 문서를 넣어주세요.

## 🤖 AI 참조 방법
파일을 업로드하신 후 저에게 알려주시면, 제가 이 폴더의 내용을 분석하여 보고서 및 발표 자료 생성 시 디자인 스타일을 맞추도록 노력하겠습니다.
